## How to
In a terminal (ex: bash / sh / xterm ...) :

* Step 1
```shell
git clone git@github.com:K0n3k/blog_openclassrooms.git
cd blog_openclassroms
composer install
``` 
* step 2

    execute blog.sql

* step 3

* step 4
